## My Todo App Setup

- npx react-native init todonative
- npm start (metro)
- npm run ios
- npm run android

Libraries

- React Native Vector Icons >> (npm install --save react-native-vector-icons)
- React Native AsyncStorage >> (npm install @react-native-async-storage/async-storage)
